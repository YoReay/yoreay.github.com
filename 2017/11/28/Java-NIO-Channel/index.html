Java NIO Channel

发表于 2017-11-28 | 更新于: 2017-11-28 | 分类于 后端

Java NIO的通道与流类似，但又有点不同：

  • 既可以从通道中读取数据，又能将数据写入通道。但流的读写是单向的。
  • 通道的读写是异步的。
  • 通道总是从缓冲区读取数据，或者将数据写入缓冲区。

正如上面所说，从通道读取数据到缓冲区，从缓冲区写入数据到通道。

Channel实现类

以下是Java NIO中最重要的Channel实现类：

  • FileChannel
  • DatagramChannel
  • SocketChannel
  • ServerSocketChannel

FileChannel从文件中读写数据。
DatagramChannel能够通过UDP读写网络中的数据。
SocketChannel能通过TCP独写网络中的数据。
ServerSocketChannel可以监听新进来的TCP连接，像Web服务器那样。对每一个新进来的连接都会创建一个SocketChannel。
